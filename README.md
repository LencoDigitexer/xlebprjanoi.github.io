# **`Term-Cust`**
***
The program is designed for styling the Thermux application.
With this program you can change the font, themes, greeting and ps1.
The program will be updated, new features will be added.
***
### Instailing
* `apt update`
* `apt upgrade`
* `apt install git`
* `git clone `
* `cd Term-Cust`
* `sh Instailing`
***
### Launch
* `Term-Cust`
